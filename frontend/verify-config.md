# 快速验证系统配置动态加载功能

## 快速验证步骤

### 1. 初始化数据库配置

```bash
cd ../YoudaoNoteLM
go run cmd/seed_sysconfig/main.go
```

### 2. 检查数据库

```bash
mysql -u root -p20041211wzwaicjW. youdao -e "SELECT config_group, config_key, enabled FROM sys_config;"
```

**预期结果**：
- 应该看到 `asr`、`search`、`embedding` 分组的配置
- `enabled` 字段应该为 `1`

### 3. 启动服务

**终端 1 - 后端**：
```bash
cd ../YoudaoNoteLM
go run cmd/server/main.go
```

**终端 2 - 前端**：
```bash
cd ../YouDaoNoteLM_Web
npm run dev
```

### 4. 访问后台管理

1. 打开浏览器访问 `http://localhost:5173/admin`
2. 登录管理员账号
3. 点击"系统配置"标签页

### 5. 验证配置显示

**应该看到**：
- 三个分组：搜索引擎、语音识别、向量化
- 每个分组下面有配置列表
- 敏感信息显示为遮罩

### 6. 测试配置编辑

1. 点击某个配置的编辑按钮
2. 修改配置值
3. 点击保存
4. 刷新页面确认修改已保存

### 7. 测试配置动态更新

1. 在后台修改配置
2. 不重启后端服务
3. 使用相关功能
4. 确认使用新配置

## 常见问题

### 问题 1: 种子脚本运行失败

**原因**：数据库连接失败

**解决**：
```bash
# 检查 MySQL 是否运行
mysql -u root -p20041211wzwaicjW. -e "SELECT 1"

# 检查数据库是否存在
mysql -u root -p20041211wzwaicjW. -e "SHOW DATABASES LIKE 'youdao'"
```

### 问题 2: 后端启动失败

**原因**：端口被占用或配置错误

**解决**：
```bash
# 检查端口是否被占用
netstat -ano | findstr :8080

# 检查配置文件
cat ../YoudaoNoteLM/configs/config.yaml
```

### 问题 3: 前端页面无法访问

**原因**：前端服务未启动或端口错误

**解决**：
```bash
# 检查前端服务是否运行
curl http://localhost:5173

# 检查端口
netstat -ano | findstr :5173
```

### 问题 4: 配置编辑后未生效

**原因**：缓存未失效

**解决**：
1. 清除浏览器缓存
2. 等待 5 分钟（缓存过期）
3. 重启后端服务
