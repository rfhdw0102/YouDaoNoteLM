# 有道云 API Key 绑定指南

## 概述

用户需要在设置页面绑定有道云笔记 API Key，才能使用有道云笔记导入功能。

## 绑定步骤

### 步骤 1: 访问设置页面

1. 登录系统
2. 点击右上角用户头像
3. 选择"设置"选项

### 步骤 2: 选择有道云笔记标签

在设置页面中，点击"有道云笔记"标签页。

**标签页顺序**：
```
[基础模型] [搜索引擎] [语音识别] [向量化] [有道云笔记]
```

### 步骤 3: 绑定 API Key

#### 未绑定状态

如果尚未绑定有道云账号，会显示绑定界面：

1. **输入 API Key**
   - 在"API Key"输入框中输入你的有道云笔记 API Key
   - API Key 是一串字符，用于验证你的有道云账号

2. **点击绑定按钮**
   - 点击"绑定账号"按钮
   - 系统会验证 API Key 的有效性

3. **绑定成功**
   - 验证通过后，界面会显示"已绑定"状态
   - 现在可以使用有道云笔记导入功能

#### 已绑定状态

如果已经绑定有道云账号，会显示绑定状态：

- **状态显示**：显示"已绑定"和绑定状态
- **解绑按钮**：可以点击"解绑账号"解除绑定

## 获取 API Key

### 方法 1: 网页版获取

1. 访问有道云笔记网页版：https://note.youdao.com/
2. 登录你的有道云账号
3. 进入"设置"页面
4. 找到"API Key"或"开发者选项"
5. 复制你的 API Key

### 方法 2: 客户端获取

1. 打开有道云笔记客户端
2. 进入"设置" > "账户"
3. 查找"API Key"选项
4. 复制你的 API Key

### 方法 3: 开发者文档

1. 访问有道云笔记开发者文档
2. 按照文档指引获取 API Key
3. 可能需要申请开发者权限

## 常见问题

### Q1: 绑定失败怎么办？

**可能原因**：
- API Key 输入错误
- API Key 已过期
- 网络连接问题

**解决方案**：
1. 检查 API Key 是否正确
2. 重新获取 API Key
3. 检查网络连接
4. 稍后重试

### Q2: 忘记 API Key 怎么办？

**解决方案**：
1. 按照上述方法重新获取 API Key
2. 如果无法获取，联系有道云客服

### Q3: 绑定后无法导入笔记？

**可能原因**：
- API Key 权限不足
- 有道云账号未开通相关功能
- 系统配置问题

**解决方案**：
1. 检查 API Key 权限
2. 确认有道云账号状态
3. 联系系统管理员

### Q4: 如何解绑账号？

**操作步骤**：
1. 访问设置页面
2. 点击"有道云笔记"标签
3. 点击"解绑账号"按钮
4. 确认解绑

## 安全提示

### 保护你的 API Key

1. **不要分享**：不要将 API Key 分享给他人
2. **定期更换**：定期更换 API Key 以提高安全性
3. **安全存储**：将 API Key 存储在安全的地方

### 绑定安全

1. **官方渠道**：只在官方渠道绑定 API Key
2. **验证来源**：确保访问的是官方网站
3. **及时解绑**：不再使用时及时解绑

## 使用流程

### 完整使用流程

```
1. 获取有道云 API Key
2. 访问设置页面
3. 点击"有道云笔记"标签
4. 输入 API Key
5. 点击"绑定账号"
6. 绑定成功
7. 使用有道云导入功能
```

### 导入笔记流程

```
1. 打开笔记本详情页
2. 点击"导入资料"按钮
3. 选择"有道云"标签
4. 浏览有道云笔记目录
5. 选择要导入的笔记
6. 点击"导入选中笔记"
7. 等待导入完成
```

## 技术说明

### 绑定验证

系统会通过以下步骤验证 API Key：

1. **格式验证**：检查 API Key 格式是否正确
2. **有效性验证**：调用有道云 API 验证 Key 是否有效
3. **权限验证**：检查 API Key 是否有导入权限

### 绑定状态

绑定状态包括：

- **未绑定**：用户未绑定有道云账号
- **已绑定**：用户已绑定有道云账号
- **绑定失败**：绑定过程中出现错误

### 状态查询

系统会定期查询绑定状态，确保绑定仍然有效。
