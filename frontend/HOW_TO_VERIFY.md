# 如何验证系统配置动态加载功能

## 快速验证（5 分钟）

### 方法 1: 使用快速验证脚本

```bash
cd ../YouDaoNoteLM_Web
./quick-verify.sh
```

这个脚本会自动检查：
- 数据库连接
- 配置表是否存在
- 配置数据是否存在
- 后端服务状态
- 前端服务状态

### 方法 2: 手动验证

#### 步骤 1: 初始化数据库配置

```bash
cd ../YoudaoNoteLM
go run cmd/seed_sysconfig/main.go
```

**预期结果**：
```
✅ [asr/aliyun_nls] 阿里云智能语音 ASR 服务
✅ [markitdown/default] MarkItDown 文档转换服务
✅ [storage/minio] MinIO 对象存储服务

所有系统配置已写入 sys_config 表
```

#### 步骤 2: 启动服务

**终端 1 - 后端**：
```bash
cd ../YoudaoNoteLM
go run cmd/server/main.go
```

**终端 2 - 前端**：
```bash
cd ../YouDaoNoteLM_Web
npm run dev
```

#### 步骤 3: 验证后台管理

1. 打开浏览器访问 `http://localhost:5173/admin`
2. 登录管理员账号
3. 点击"系统配置"标签页

**预期结果**：
- 显示三个分组：搜索引擎、语音识别、向量化
- 每个分组有配置数量显示
- 点击分组可查看配置详情

## 故障排除

### 问题 1: 种子脚本运行失败

**原因**：数据库连接失败

**解决**：
```bash
# 检查 MySQL 是否运行
mysql -u root -p20041211wzwaicjW. -e "SELECT 1"

# 检查数据库是否存在
mysql -u root -p20041211wzwaicjW. -e "SHOW DATABASES LIKE 'youdao'"
```

### 问题 2: 后端启动失败

**原因**：端口被占用或配置错误

**解决**：
```bash
# 检查端口是否被占用
netstat -ano | findstr :8080

# 检查配置文件
cat ../YoudaoNoteLM/configs/config.yaml
```

### 问题 3: 前端页面无法访问

**原因**：前端服务未启动或端口错误

**解决**：
```bash
# 检查前端服务是否运行
curl http://localhost:5173

# 检查端口
netstat -ano | findstr :5173
```

### 问题 4: 配置编辑后未生效

**原因**：缓存未失效

**解决**：
1. 清除浏览器缓存
2. 等待 5 分钟（缓存过期）
3. 重启后端服务
