# 在哪里绑定有道云 API Key

## 快速答案

用户需要在**设置页面**的**有道云笔记标签页**绑定 API Key。

## 详细步骤

### 1. 访问设置页面

- 登录系统
- 点击右上角用户头像
- 选择"设置"

### 2. 找到有道云笔记标签

在设置页面中，找到并点击"有道云笔记"标签：

```
[基础模型] [搜索引擎] [语音识别] [向量化] [有道云笔记]
                                                      ^
                                                      点击这里
```

### 3. 绑定 API Key

在有道云笔记标签页中：

1. **输入 API Key**：在输入框中输入你的有道云 API Key
2. **点击绑定**：点击"绑定账号"按钮
3. **完成绑定**：等待系统验证并完成绑定

## 如何获取 API Key

### 方法 1: 网页版

1. 访问 https://note.youdao.com/
2. 登录账号
3. 进入设置
4. 找到 API Key 选项
5. 复制 API Key

### 方法 2: 客户端

1. 打开有道云笔记客户端
2. 进入设置 > 账户
3. 查找 API Key 选项
4. 复制 API Key

## 常见问题

**Q: 绑定失败怎么办？**
A: 检查 API Key 是否正确，重新获取 API Key，检查网络连接。

**Q: 忘记 API Key 怎么办？**
A: 按照上述方法重新获取 API Key。

**Q: 如何解绑账号？**
A: 在设置页面的有道云笔记标签页，点击"解绑账号"按钮。

## 下一步

绑定成功后，就可以：

1. 打开笔记本详情页
2. 点击"导入资料"按钮
3. 选择"有道云"标签
4. 浏览并导入有道云笔记

## 总结

绑定有道云 API Key 的位置：
- **页面**：设置页面
- **标签**：有道云笔记
- **操作**：输入 API Key → 点击绑定

绑定成功后即可使用有道云笔记导入功能。
